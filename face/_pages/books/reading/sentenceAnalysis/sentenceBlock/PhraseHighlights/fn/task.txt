У меня есть приложение написанное на React. Компонент SentenceBlock получает данные предложения из переменной sentence, которые имеют тип ChapterTextStructurePopulated.Sentence. И в SentenceBlock отрисовываются части этого предложения: слова, пробелы, знаки препинания. Мне нужно чтобы некоторые слова входящие во фразы были подствечены фоновыми прямоугольниками. Для этого я подготовил компонент PhraseHighlights где и будут располагаться прямоугольники по размерам совпадающие с размерами слов.

В компоненте PhraseHighlights размеры подсвечивающих прямоугольников вычисляются функцией useGetHighlights и затем отрисовываются с абсолютным позиционированием.

В функции useGetHighlights получается текущее предложение:
const selectedSentence = useGetSelectedSentence()

Из текущего предложения нужно взять все фразы:
selectedSentence.phrases
и перебрать их. В каждой фразе есть свойство wordIds — это массив идентификаторов слов входящих во фразу. Сами слова содержатся в массиве parts переменной selectedSentence и имеют тип word. Тебе нужно узнать идентификаторы всех слов входящих во фразу и получить их размеры чтобы затем отрисовать подсвечивающие прямоугольники в компоненте PhraseHighlights. Сделай размеры подсвечивающих прямоугольников немного большие, чем размеры слова. Просто сделай внешние переменные где будут задаваться значение увеличения чтобы дальше я смог самостоятельно указать разницу между настоящим размером и увеличенным для левой, правой, верхней и нижней стороны подствечивающего прямоугольника.

Если идентификаторы слов фразы имеют тип idle, то подствечивающему прямоугольнику поставить класс CSS word-highlight--idle. В противном случае класс word-highlight.
