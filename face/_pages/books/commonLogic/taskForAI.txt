@textIntoChapterStructure.ts Функция textIntoChapterStructure принимает текст. Например
`Alice was very tired: she had the book. “What is the use of a book,” thought Alice “without conversations?”

So she was confused.
White Rabbit ran by her.`

В ответ функция должна вернуть массив как в переменной expectedResult.

expectedResult — это массив типа ChapterTextStructure.Chapter. Каждый элемент массива состоит из одного из следующих типов данных:

Тип предложения:
type Sentence = {
	t: 'sentence' // type: sentence
	translation: null | string
	parts: SentencePart[]
}

Тип пробела между предложениями:
type Space = {
	t: 'space' // type: space
}

Тип пунктуации:
type Punctuation = {
	t: 'punctuation' // type: punctuation
	v: string // value
}

Тип возврата каретки:
type CarriageReturn = {
	t: 'carriageReturn' // type: carriage return
}

Каждое предложение может состоять из слов (ChapterTextStructure.Word), пробельных знаков (ChapterTextStructure.Space) и знаков пунктуации (ChapterTextStructure.Punctuation).

Нужно определить что является предложением. Например предложение заканчивается точкой, восклицательным или вопросительными знаками:
Alice was very tired: she had the book.
How brave they’ll all think me at home!
Would the fall never come to an end?

Есть более сложные предложения где точка, восклицательный или вопросительными знак есть внутри предложения в виде прямой речи. Воспринимай это как одно большое предложение:
There was nothing so very remarkable in that; nor did Alice think it so very much out of the way to hear the Rabbit say to itself, “Oh dear! Oh dear! I shall be late!” (when she thought it over afterwards, it occurred to her that she ought to have wondered at this, but at the time it all seemed quite natural); but when the Rabbit actually took a watch out of its waistcoat-pocket, and looked at it, and then hurried on, Alice started to her feet, for it flashed across her mind that she had never before seen a rabbit with either a waistcoat-pocket, or a watch to take out of it, and burning with curiosity, she ran across the field after it, and fortunately was just in time to see it pop down a large rabbit-hole under the hedge.

Ещё пример большого предложения:
“Well!” thought Alice to herself, “after such a fall as this, I shall think nothing of tumbling down stairs!

Учитывай, что если между предложениями есть пробел, то его тоже нужно ставить в структуру.

Знаками пунктуации могут быть все знаки кроме слов, пробелов, символа возврата каретки. Если подряд идут несколько знаков пунктуации, то их можно поставить в один объект. Например:
{
	t: 'punctuation',
	v: ',”',
}

Если слово или предложение начинается с новой строки, то должен быть только один объект возврата каретки. Если между предложениями есть пустая строка, то должно быть 2 объекта возврата каретки.

Напиши тесты чтобы проверить работоспособность функции.
